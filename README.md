# Lomilab

## Introduction
- Lomilab is a collection of projects that are created to help developers to create extremely fast model AI, ML lightweight to Zero-threshold deployment. Open-source 
- Provide a user-friendly API for tracking training and analyzing data such as parameters, neural,  tracking parameter, metric, ...

1. Analysis
   - Recording the hyperparameters, metrics, and model parameters
   - Chart types such as line, bar, scatter, and histogram
   - Support Image 3D for Neural Network
   - Metadata for each chart, types, and data
   - Automatically save the data to the database, loggings, and filter
   - Local Push your Model, Datasets to the server

2. Framework Integration
    - Deep Learning: Pytorch, Tensorflow, Keras, FastAI, Huggingface, ...
    - Machine Learning: Scikit-learn, XGBoost, LightGBM, Catboost, ...
    - Data Processing: Pandas, Numpy, Dask, ...
3. Experiments Model, Datasets
    - 1 Dashboard to manage all your experiments, multiple projects
    - 1 Dashboard to manage all your datasets, multiple projects
4. Collaborator:
    - Share your experiments, datasets with your team
    - Share your model, datasets with the community
5. Shared:
    - Copy, Clone and Send URL like Git to share your each experiment, datasets
6. Support hosting on the cloud
    - Support offiline, online, and hybrid
